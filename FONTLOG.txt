FONTLOG for the Aleo font

This file provides detailed information on the Aleo Font Software.
This information should be distributed along with the Aleo fonts
and any derivative works.

Basic Font Information

Aleo is a Unicode typeface family that supports all languages that
use the Latin script and its variants, and could be expanded to support other
scripts.

This release contains an OFL release of Aleo intended for inclusion in Google
Fonts and other web projects.

Documentation can be found at https://github.com/AlessioLaiso/aleo/
